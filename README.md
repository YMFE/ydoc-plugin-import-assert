# ydoc-plugin-import-assert
Import js and css files in the page.

## Install
```
npm install ydoc-plugin-import-assert
```
## Usage

In the ydoc config file, configuration:

```
{
  plugins: [{
    name: 'import-assert',
    options: {
      css: 'custom.css',
      js:  'custom.js'
    }
  }]
}
```
